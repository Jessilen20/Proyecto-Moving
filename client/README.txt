*Descripcion de la app
Es una p√°gina web que muestra un catalogo de academias deportivas y culturales; se desarrollo para que pueda ser utilizado por instituciones educativas y puedan 
organizar mejor sus inscripciones.

Nota: El proyecto aun esta en desarrollo, por tal motivo el usuario objetivo serian los profesores de la institucion educativa porque tienen accesos a editar y eliminar 
cualquier inscripcion. Para una segunda fase del proyecto, se agregaria que el usuario logueado solo pueda ver la lista de inscritos pero exclusivamente pueda editar y eliminar las creadas
por el mismo usuario.

La pagina principal muestra el catalogo de academias ofertadas y al ingresar a cada una se observa el detalle de las mismas.


*Requisitos para iniciar el cliente
1. Instalar npm y las dependencias necesarias que se encuentran en el package.json

2. Se inicia el proyecto con npm start


*Dependencias usuadas:
-Bootstrap y React Bootstrap para el manejo de estilos
-Axios  para el llamado a las apis
-React Router Dom para el manejo de rutas internas
-Formik y Yup para el manejo de formularios y validaciones
